# PhotoStore PSO

PhotoStore (PSO) is a simple photo editing software project developed with Processing during my first-year computer project course.

## Project Introduction

This project was created as an undergraduate programming project.  
The goal was to design a small desktop image-editing application with a custom graphical user interface.

## Features

- Drawing on canvas
- Eraser tool
- Text-related function
- Decorative stamp  icon tools
- Selection-related function
- Basic image editing interaction
- Custom hand-drawn style user interface

## Tech Stack

- Processing
- Java-based graphical programming
- Local image assets for interface design

## Project Structure

- `Initial_and_Settings.pde` – initialization and basic settings
- `Main_Functions.pde` – main functional implementation
- `main.pde` – visualize interface
- `data` – image assets and project resources

## How to Run

### Option 1 Run in Processing
1. Install Processing
2. Open the project folder
3. Open the `main.pde` file in **Processing**
4. Click Run

### Option 2 Run the Windows executable
 Open the Windows release folder and run the `.exe` file after packing it.
